Installation -> Beginning

Welcome to the Omni Help Desk installation process.

This install wizard will guide you through the installation process. If you already have db_config.php setup in your OHD directory you will be redirected directly to Step 2.

Step 1 - If you are not running your own server or if there are alot of other users who access this server we DO NOT recommend you to set the permissions of the ohd directory to be readable and writable by the user of the web server.

We say this because everyone on this server will be able to read the config file which reveals your mySQL username and password. If you are not using your own personal server DO NOT set the OHD directory to be readable and writable from the webserver. Create your db_config.php manually and upload it into the OHD folder. There is an example db_config.php.exm.
